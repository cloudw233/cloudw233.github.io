如何与你的同学玩MC

2022-01-08 3 min read

与你同学快乐游玩MC

你是否有以下困惑

  • 啊啊啊啊这个怎么启动！
  • 啊啊啊啊这个好难操作
  • 为什么又下载失败了啊啊啊啊

阅读完这篇文章，或许能解开你心中的困惑

双击运行下列文件

  • dotnet.exe
  • jdk17.msi

你需要运行它们并按引导一步步安装成功

创建一个帐户

打开文件夹里的HMCL.exe

  • 点击图示按钮
  • 然后创建离线帐户
  • 点击创建后随便输入一个名称 一定是英文！，然后点登录

完成以上步骤后，请看下一章

安装游戏

你想和你同学玩的话，你白嫖他做好的整合包更方便，对吧？下载整合包：
1.18.1-build-with-love

将你下载到的这个文件拖入启动器界面内，等待安装完成

进入游戏

进入游戏后

  1. 点击多人游戏
  2. 选择第一个或者第二个服务器进入
  3. 按照服务器提示注册

提醒：别把你密码忘了

  1. 用鼠标右键点击那个写着1.18生存的玩家
  2. 再与我联系，我一般会立马上线
  3. 愉快玩耍！

游戏教程

这里仅列出基本的快捷键，具体教程请自行搜索

  • F3系列：
  1. F3+B:显示碰撞箱
  2. F3+D:清空聊天栏
  3. F3+G:显示区块边界
  4. F3+H:显示一些高级的东西（耐久度、英文ID等）（我习惯把木镐耐久度用到最低然后把它丢进熔炉里）
  5. F3+F4:把创造模式变成旁观者模式，或把其他模式变成创造模式
  6. F3+P:挂机神器
  7. F3+Q:F3快捷键教学
  8. F3+Esc:暂停游戏但不显示菜单
  • Ctrl系列：
  1. Ctrl+Q:把一叠东西全丢出
  2. Ctrl+鼠标中键:获得带NBT的方块（比如里面有东西的箱子）
  3. 物品栏：双击左键：收集所有物品栏内同类物品（最多一组）
  • Shift系列：
  1. Shift+单击"X"：清空物品栏（仅限创造）
  2. Shift+单击物品：获得一组物品
  3. Shift/Ctrl+数字键+鼠标悬停于物品上：在快捷栏对应位置获得一组相应物品
  4. Shift+左键单击合成表中的产物：放上能制作最多产物的原料
  • 杂项
  1. c+数字键：保存快捷栏
  2. x+数字键：加载快捷栏合成：
  3. 鼠标中键：获得方块和刷怪蛋
  4. F3：获取调试屏幕（指看坐标）
  5. Ctrl：强制进入疾跑状态
  6. L：显示进度数字键：把物品放到快捷栏上对应位置
  7. /：直接输命令，不用按T
